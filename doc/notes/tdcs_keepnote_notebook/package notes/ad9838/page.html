ZDmax = Dmin + 2J_t + T_t = 3.9 + 2*0.1 + .31 = 4.4mm

ZDmax is the max landing pattern dimension ZD = ZE for this package

Dmin is the package minimum width Dmin = 3.9mm

J_t is  the minimum dimension of the toe fillet for a succesful solder joint J_t > 0.1mm

T_t is the tolerance of the toe fillet tollerance, which is the sum of the tollerances for the placement, board, and component. Tt ~ .31mm



GDmin >= ADMax + 2*C_LL = 2.25mm + 0.15mm = 2.4mm

GDmin is the minimum distance between the inside edges of opposite traces

C_LL is the corner clearance, set to 0.1 mm

ADMax is the max width across a side of pads, = Lead_pitch * (# of leads -1) +pad width = 0.5mm (4) + .25 = 2.25mm



Y = (ZDmax - GDmin)/2 = 1mm

Y is length of pad


D2 = GD_min - 2CPL = 2.4mm - 2*.25mm = 1.9mm

D2 is the inner thermal padle dimension, Cpl is the clearance bretween the inner edge of the pad and the exposed pad, set at .25mm
